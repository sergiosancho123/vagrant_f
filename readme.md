<!--VAGRANT: ¿QUE ES Y PARA QUE SIRVE? (para mi)-->
Vagrant es una herramienta de código abierto que se utiliza para crear y compratir el desarollo con otros entornos virtuales de una manera más sencilla.

Una vez instalado el ejecutador de vagrant en windows y ejecutarlo, los comandos que utilizo para instalar vagrant en mi consola de VISUAL CODE son: para crear un fichero llamado "vagrantfile" es un touch "nombre_fichero" que en este caso es Vagrantfile --> - touch Vagrantfile.

También dentro de dicho archivo necesitamos para la configuracion meter algo dentro, que ya lo hemos metido.

-config.vm.box = "windows/AMD64" es la imagen.
